240 番茄钟软件 Pomodairo

若不知道番茄钟是什么意思，请先了解什么是番茄工作法？
（在电脑上工作做事，用番茄工作法是个挺好的时间管理方式。一是促使工作处理的更有效率，二是提醒自己定时休息一下眼睛、脖子、鼠标手，站起来活动一下筋骨。小习惯能有效保护身体。）

 

Pomodario 最新版本是 Pomodairo 1.9 （2013-06-04）

截图1 via Pomodario

界面简洁，易于使用，能完全符合番茄工作法的需求。而且它是一款免费开源的应用软件。
功能大概包括：

增加/修改任务；
支持自定义番茄时间。短间隔休息时长、几个番茄时间为一个长周期，一个长周期后的长间隔休息时长；
自带提示音效，同时支持自定义提示音；
支持计时窗口始终置顶
支持任务列表信息导出（以文本格式复制到剪贴板）
简单的统计

1、添加任务
点击“Task List”打开任务列表，再点击右下角“Add New”添加新任务，在弹出的窗口中，左侧文本框输入任务名称，右侧文本框输入估计要耗用的番茄数，Enter回车键确认添加任务，或按Esc键取消添加。

 

2、任务列表说明
Name 任务列表的名称
E（estimated)是预估耗用番茄数
“Add New”新建任务时，任务名称右边输入的数字
P（Pomodoros）是实际番茄数
已经花在该活动的番茄数，即当前任务中，点击“Start Timer”开始计时直至该番茄时间结束，当前任务的实际番茄数加1。
U（unplanned）是“计划外”
在这样任务上添加了几个计划外任务
I（interruptions）干扰
在番茄计时期间受到干扰，但及时排除的项目。

 

3、开始执行任务，以及任务的变化或修改
Start Timer 开始计时
在任务列表中，选中要执行的任务。（倒计时数字下会显示当前的任务名称，如图3中的“Make release”）点击“Start Timer”开始当前任务的一个番茄时间。中途不可暂停，点击“Stop Timer”是停止计时，放弃当前番茄时间。
Unplanned 添加计划外任务
比如我先建立了一个“读书-abc”的任务，然后我又打算在每读完一章写读书笔记做整理总结，我就选中“读书-abc”的任务，点击“Unplanned”添加一个计划外的“读书笔记-abc”。再引用网友的一个例子，假设我先建立了一个任务叫“发邮件”，但开始工作时又发现需要先把方案整理好，因此选中“发邮件”活动，点击“Unplanned”添加一个“整理方案”的衍生任务。
点击“Unplanned”按钮，会弹出和“Add New”一样的添加任务小窗口，填写任务名称和所需番茄数，回车确认。这时会看到任务列表里增加了一项任务，并当前任务的“U”一栏的数字会加1，表示这个活动衍生出了一个计划外任务。
Interruption 打岔记录被打岔，即已排除的干扰，仍旧继续当前任务。
Edit 修改
在已有的任务上点击鼠标右键，选择“Edit”可以修改任务的各项属性。

 

4、设置提示音
通过菜单 Menu→Config→Sounds 打开提示音设置窗口
Volume（音量），调整提示音的声音大小
Alarm Sound（报警声），即一个番茄时间结束时的提示音。
Clock Sound（计时声）， 即一个番茄时间开始计时的提示音。
不喜欢自带的提示音，可以通过以 Alarm Sound 和 Clock Sound 来自定义提示音。点击“Browse”打开文件对话框，选择mp3格式的声音文件。
Sound enabled（启用音效），选中即会有提示音，反之就没有提示音。
Fade ticking（计时声淡出），选中即计时声响起后逐渐淡出至无声，反之计时声会一直滴答滴答响。

 

5、修改番茄时间
通过菜单 Menu→Config→Timing 打开时间设置窗口
Pomodoro length 一个番茄时间的长度，默认为25分钟
Short break length 一个番茄时间之后的短时休息的时间长度，默认为5分钟
Long break interval 设定几个番茄时间之后进行一次长时休息，默认是4个。
Long break length 若干个（默认4个）番茄时间之后的长时休息的时间长度，默认为20分钟
